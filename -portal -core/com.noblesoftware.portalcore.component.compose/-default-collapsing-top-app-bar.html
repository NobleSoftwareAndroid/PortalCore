DefaultCollapsingTopAppBar

@Composable
fun DefaultCollapsingTopAppBar(modifier: Modifier, title: String, plain: Boolean? = false, canBack: Boolean? = false, canClose: Boolean? = false, onBackClick: () -> Any? = {}, onCloseClick: () -> Any? = {}, navigator: NavHostController? = null, scrollBehavior: TopAppBarScrollBehavior? = null, containerColor: Color? = null, titleTextColor: Color? = null, actions: @Composable () -> Unit? = {}, tabs: @Composable () -> Unit? = {})

The DefaultTopAppBar composable is used to create a top app bar (also known as a toolbar) with customizable properties. It provides consistent navigation and action controls aligned with Material Design principles.

Parameters

Author

VPN Android Team

Since

Parameters

modifier

A modifier that can be used to customize the appearance of the top app bar.

title

The title text to display in the app bar.

plain

An optional boolean indicating whether the app bar should have a plain appearance (without elevation or background). Defaults to false.

canBack

An optional boolean indicating whether a back navigation button should be displayed. Defaults to false.

canClose

An optional boolean indicating whether a close button should be displayed. Defaults to false.

onBackClick

A lambda function to execute when the back button is clicked.

onCloseClick

A lambda function to execute when the close button is clicked.

navigator

An optional NavHostController for handling navigation actions.

actions

An optional composable lambda that defines additional action icons or buttons to display in the app bar.
